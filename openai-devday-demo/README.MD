## 环境 set up
安装 openai
```
$ pip install openai

$ pip freeze | grep openai >> requirements.txt
```

安装依赖包
```
pip install pydub
pip freeze | grep pydub >> requirements.txt

```

按装ffmpeg 、mpv播放声音

```commandline
 播放声音
 brew install ffmpeg
 播放声音流
 brew install mpv
```
